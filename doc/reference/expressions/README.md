# Expressions

Expressions in Typhon evaluate to a value. This section covers the various types of expressions available in the language.

- [Literals](literals.md): Basic values and data structures.
- [Operators](operators.md): Arithmetic, logical, and comparison operators.
- [Null Safety](null_safety.md): Tools for handling optional values.
- [Pipe Operator](pipe_operator.md): Functional piping of values.
- [Placeholders](placeholders.md): Concise anonymous functions.
- [Comprehensions](comprehensions.md): Generating collections and control flow expressions.
